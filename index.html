4. Dicas Importantes

  • <p>: Cria blocos de texto com margem antes e depois.
  • <br>: Quebra de linha no meio do parágrafo, sem margem.
  • Use <p> para separar ideias; use <br> para forçar uma quebra visual.
  • <b> e <i> são usados apenas para estilizar o texto dentro dos parágrafos.
