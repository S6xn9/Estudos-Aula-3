Inserindo Símbolos Especiais e Emojis em HTML

1. Por que usar entidades HTML?

Alguns caracteres especiais podem causar confusão com a própria sintaxe do HTML (como <, >, &, etc.). Por isso, usamos entidades HTML para representá-los de forma segura.

2. Exemplos de Símbolos Especiais

  • &lt; → <
  • &gt; → >
  • &amp; → &
  • &copy; → ©
  • &euro; → €
  • &yen; → ¥
  • &reg; → ®

Esses códigos são chamados de entidades HTML e começam com & e terminam com ;

3. Inserindo Emojis 😊

Você pode inserir emojis diretamente no código HTML, desde que o documento esteja usando UTF-8 no charset (como no <meta charset="UTF-8">).

Exemplo de emojis inseridos diretamente:

  • 😀 😎 🤖 🎉 ❤️
  • 🔥 📚 💻 ✅ 🚀

Esses emojis são compatíveis com a maioria dos navegadores e sistemas modernos.

4. Observações Importantes

  • Prefira sempre usar charset="UTF-8" no <head> para suportar emojis e caracteres especiais.
  • Nem todos os emojis ou símbolos aparecem igual em todos os sistemas (Windows, Android, iOS...)
  • Use entidades HTML quando quiser garantir a exibição correta de caracteres reservados.
